Software Engineering Projects

FitForge

Developed a fully functional website that generates currated, personal, and fun training programs for newer gym attendees. The web application was capable of querying data from the mySQL back-end and presenting the data to the user in the front-end. Also helped develop an algorithm that pulled specific workouts for the user and populated a table with the associated target weight, repetitions, and R.P.E.

Weather App

Developed a C++ program that can accurately display the local weather depending on the user's location. This works by accessing the OpenWeatherAPI and collecting the data into a XML file based on the coordinates of the user. It then can accurately parse the data and present the data in a graphical user interface that i implemented using the Qt C++ GUI framework.
